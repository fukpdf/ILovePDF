How to Crop an Image Online Free — Precise Image Cropping

Cropping an image used to mean opening Photoshop, or fighting with phone gallery sliders. The Crop Image tool lets you crop to precise percentage values straight from your browser — no software, no signup.

Use it for cropping product photos, profile pictures, social-media banners, or anything else where exact dimensions matter.

Step-by-step: how to use Crop Image

  1. Open the tool. Visit ilovepdf.cyou/crop-image.
  2. Upload your image. Drag and drop a .jpg, .jpeg, .png or .webp file.
  3. Set crop offsets. Enter X offset, Y offset, width and height as percentages of the original image.
  4. Process. The tool slices out the requested region.
  5. Download. Save the cropped image as ILovePDF-crop-image.png.

Why use Crop Image?

  • Pixel precision. Percentage-based cropping is portable across image sizes and resolutions.
  • Lossless. Cropping doesn't recompress — the kept region retains original quality.
  • Browser-based. Works on phone, tablet or laptop without installing software.
  • Free. No watermark, no signup, unlimited crops.

Pro tips for the best results

1. Use 100% width and height for centring

Set width=100 and height=100 with X/Y offsets to use the full canvas as a starting point.

2. Combine with resize

After cropping, run through Image Resize for exact pixel dimensions.

3. Remove background after

For product photos, follow with Background Remover for a clean transparent result.

4. Apply filters last

Image Filters work best on the final cropped composition.

Frequently asked questions

Will it preserve image quality?

Yes. Cropping is lossless — the kept region matches the original pixel-for-pixel.

Can I crop multiple images?

One image per run. Repeat for batch cropping.

Does it work for very large images?

Yes, up to the 100 MB upload limit.

Can I undo a crop?

Cropping outputs a new file — the original is preserved on your device.
