How to Crop a PDF Online — Trim Margins for Free

  • Secure processing
  • No installation required
  • Files auto-deleted after 10 minutes

A PDF with bloated white margins or jagged scan borders looks unprofessional and wastes paper when printed. Cropping tightens the visible content area without ever touching the original underlying text or images.

This guide shows you how to use the Crop PDF tool to trim every page to a clean, consistent rectangle in just a few clicks.

Step-by-step: how to use Crop PDF

  1. Open Crop PDF. Go to ilovepdf.cyou/crop-pdf.
  2. Upload your PDF. Drop the file in. The tool processes all pages with the same crop values.
  3. Set crop percentages. Enter how much of each side to trim — Left, Right, Top and Bottom — as a percentage (e.g. 5 for 5%).
  4. Process and preview. Click Process. The cropped PDF generates in seconds.
  5. Download. The result downloads as ILovePDF-crop.pdf. Re-process if you need different values.

Why use Crop PDF?

  • Cleaner output. Removes scan margins, watermarked footers, and excessive whitespace for a professional look.
  • Smaller file size. Pages with smaller cropbox areas often render and download faster, especially when re-printed.
  • Print-friendly. Tighter margins mean less wasted ink and paper when printing PDFs.
  • Consistent with PDF standards. Crop changes the page CropBox, which every PDF reader respects.

Common use cases

For Academic papers

Trim the wide journal margins of a research PDF before printing to save paper and ink.

For Scanned books

Remove the dark scanner shadow on the gutter side of every page in a scanned book.

For Mobile reading

Crop unnecessary whitespace so text renders larger on a phone or e-reader screen.

Pro tips for the best results

1. Start small

Try 3–5% per side first. Aggressive crops can cut into actual content.

2. Crop after rotating

If the page needs rotation, do that first so your crop directions stay correct.

3. Re-export images if cropping a lot

If you crop heavily, follow up with Compress PDF to actually reduce the file size.

4. Inspect before sharing

Open the cropped result in your normal viewer to confirm nothing important was clipped.

Frequently asked questions

Does cropping delete content?

No. Cropping changes only the visible area (CropBox). The underlying content is preserved and could theoretically be restored.

Can I crop different pages by different amounts?

The tool applies one crop to all pages. Split the PDF first if you need per-page values.

Will the crop survive in other PDF readers?

Yes — any spec-compliant viewer respects the new CropBox.

Why is my file size still large after cropping?

CropBox doesn't remove image data. Run Compress PDF to reduce size.
