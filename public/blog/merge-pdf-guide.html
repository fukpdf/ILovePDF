Step-by-step: how to use Merge PDF

  1. Open the Merge PDF tool. Go to ilovepdf.cyou/merge-pdf. You don't need to sign in for files under 100 MB.
  2. Upload your PDFs. Drag and drop two or more PDF files onto the upload area, or click to browse. You can select many files at once.
  3. Reorder the files. Drag the file thumbnails into the exact sequence you want them stitched together. The merged PDF respects this order.
  4. Process the merge. Click Process Files. Our server combines every page in order using pdf-lib and prepares your download.
  5. Download the result. The merged file downloads as ILovePDF-merge.pdf. You can re-download from the result screen if your browser blocked the first one.

Pro tips for the best results

1. Order before you upload

The merged PDF follows the exact upload order, so rename your files 01-cover.pdf, 02-summary.pdf etc. for predictable sequencing.

2. Compress after merging

A merged file with lots of images can be huge. Run the result through Compress PDF before emailing.

3. Unlock encrypted PDFs first

Password-protected PDFs can't be merged directly. Run them through Unlock PDF first if you own them.

4. Standardise orientation

Mixing portrait and landscape pages can look messy when printed. Use Rotate PDF on the odd ones first.
