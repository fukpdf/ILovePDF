Frequently asked questions

Will it extract every table perfectly?

Most clean tables come through accurately. Heavily merged cells or no-grid layouts may need cleanup.

Can I extract tables from a scanned PDF?

Run OCR first using OCR PDF. Otherwise the tool sees pixels, not numbers.

How are formulas handled?

PDFs don't store formulas — only their computed values. You'll need to re-enter formulas in Excel.

Does each PDF page become one sheet?

Tables are merged into the same sheet by default for easier analysis.
