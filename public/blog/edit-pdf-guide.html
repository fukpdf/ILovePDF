You don't always need a $20/month PDF editor just to add a single sentence to a contract or fix a typo on a flyer. Light PDF editing — text overlays, notes, simple annotations — should be free and instant.

The Edit PDF tool lets you stamp text on any page of a PDF directly from your browser. Here's how.

Step-by-step: how to use Edit PDF

  1. Open the editor. Visit ilovepdf.cyou/edit-pdf.
  2. Upload your PDF. Drag and drop the file you want to edit.
  3. Enter the text. Type the text you want to add, then set X / Y position (as a percentage), font size and target page.
  4. Process. The tool overlays your text onto the chosen page(s) without modifying the original content.
  5. Download. Save the result as ILovePDF-edit.pdf. The original is untouched.

Pro tips for the best results

1. Use percentage coordinates

X/Y positions are 0–100% — easier to predict than raw points across different page sizes.

2. Test on one page first

Run a single-page edit before applying to "all" to avoid mass typos.

3. Combine with Sign PDF

For signature lines, use Sign PDF instead — it's optimised for that.

4. Watermark with Watermark PDF

For diagonal watermarks, use the dedicated Watermark PDF tool.

Frequently asked questions

Can I edit existing text in the PDF?

No. The tool overlays new text on top — it doesn't modify the underlying text. For full editing, convert to Word with PDF to Word.

How precise is the placement?

Coordinates use percentages (0–100), so placement is consistent across page sizes.

Can I add images?

The current tool focuses on text. Image overlays are on the roadmap.

Does it support multiple text boxes?

One overlay per run — process the PDF multiple times for multiple text additions.
