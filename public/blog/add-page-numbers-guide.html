How to Add Page Numbers to a PDF Online Free

Long PDFs without page numbers are unnavigable. Try discussing "page 47" of a 120-page report with a colleague when neither of you knows which physical page that corresponds to. Page numbers fix that in seconds.

The Add Page Numbers tool stamps clean, customisable page numbers on every page. Here's the walkthrough.

Step-by-step: how to use Add Page Numbers

  1. Open the tool. Visit ilovepdf.cyou/add-page-numbers.
  2. Upload the PDF. Drop the file. Multi-page documents are required (page numbers don't make sense for one page!).
  3. Pick the position. Choose from 6 positions — bottom centre/left/right or top centre/left/right.
  4. Set the starting number. Default is 1, but you can start anywhere — useful when adding to the second part of a multi-volume document.
  5. Process and download. The numbered PDF saves as ILovePDF-page-numbers.pdf.

Pro tips for the best results

1. Use bottom-centre for most cases

Bottom centre is the universal default for books, reports and academic papers.

2. Use bottom-right for spiral-bound

For documents that'll be spiral-bound on the left, bottom-right keeps numbers visible.

3. Add page numbers last

Apply numbering after merging or reordering — adding earlier means renumbering later.

4. Combine with watermark

For draft documents, add both page numbers and a DRAFT watermark.

Frequently asked questions

Can I skip page numbering on certain pages?

Currently the tool numbers every page. Split off pages you want unnumbered before processing.

Can I use Roman numerals?

Arabic numerals only at the moment. Roman numeral support is planned.

Will the numbers be in colour?

Black is the default and works for every layout. Colour customisation is on the roadmap.

Does it overwrite existing page numbers?

It adds a new layer — existing numbers from the original PDF stay where they are.
