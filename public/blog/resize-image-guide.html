How to Resize an Image Online — Free Resize Tool with Presets

Sometimes an image is too big — for an upload limit, an email, a website. Sometimes it's the wrong shape — square for Instagram, widescreen for YouTube. Resizing handles both at once.

The Image Resize tool lets you scale any image to a preset (1:1 square, 16:9 widescreen, A4, HD) or to exact custom pixel dimensions.

Step-by-step: how to use Image Resize

  1. Open the tool. Visit ilovepdf.cyou/resize-image.
  2. Upload your image. Drag and drop a .jpg, .jpeg, .png or .webp file.
  3. Pick a preset or Custom. 1:1 (1080×1080), 16:9 (1920×1080), A4 (2480×3508), HD (1920×1080), or Custom.
  4. Set custom dimensions (if Custom). Enter exact width and height in pixels.
  5. Process and download. Save the resized image as ILovePDF-resize-image.png.

Pro tips for the best results

1. Crop first for aspect ratio

Use Crop Image first if your source aspect ratio doesn't match the target.

2. Don't upscale heavily

Going from 100×100 to 4000×4000 will look pixellated. Resize within reasonable bounds.

3. Combine with filters

Apply Image Filters after resizing for best visual consistency.

4. Compress for web

After resizing, lighten the file with image compression workflows.

Frequently asked questions

Will it stretch my image?

Yes — exact width/height resizes ignore the original aspect ratio. Crop first to maintain proportion.

Can I batch resize?

One image per run. Repeat for batches.

What's the max output size?

Up to roughly print resolution (4000×4000+). Very large outputs may take longer.

Does it preserve transparency?

Yes for PNG inputs/outputs. JPG outputs flatten transparency to white.
