PDF Workflow Builder — Chain Multiple Operations in One Pass

Most PDF prep involves the same sequence of steps every time — compress, watermark, add page numbers, then send. Doing them one at a time means uploading and downloading the same file three or four times. The Workflow Builder chains those operations into a single one-click pipeline.

The Workflow Builder tool lets you stack up to three operations on one upload. Here's how.

Step-by-step: how to use Workflow Builder

  1. Open the tool. Visit ilovepdf.cyou/workflow-builder.
  2. Upload your PDF. Drop the file you want to process through multiple operations.
  3. Pick Step 1. Choose from Compress, Rotate (90/180), Watermark, Add Page Numbers, or Sign. Provide a text value for Watermark/Sign.
  4. Optionally pick Steps 2 and 3. Stack up to two more operations to run in order on the same file.
  5. Process and download. The tool runs all three steps on the upload in sequence and gives you the final PDF.

Pro tips for the best results

1. Order matters

Compress last to ensure the smallest file size — earlier operations might add overhead.

2. Test each step alone first

Verify each operation works on the file individually before chaining.

3. Save the original

Workflows are destructive across multiple steps. Always keep an unmodified copy of the source file.

4. Combine with merge

Run the workflow on the merged output of Merge PDF for batch processing.

Frequently asked questions

How many steps can I chain?

Up to 3 operations per workflow run.

Can I save a workflow as a preset?

Saved presets are on the roadmap. For now, set it up each time.

Does each step run on the output of the previous?

Yes — operations are sequential, with each step receiving the previous step's output.

Can I run different operations than the listed ones?

Currently the listed five — Compress, Rotate, Watermark, Page Numbers, Sign. More operations are planned.
