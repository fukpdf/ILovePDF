How to Rotate PDF Pages Online — Free & Permanent

Scanning a stack of papers always seems to produce at least one sideways page. Most PDF viewers let you rotate temporarily, but the file itself stays wrong, so the next person you send it to sees the same crooked page.

The Rotate PDF tool fixes the orientation permanently — saving the rotated version as a fresh PDF you can share with anyone. This guide walks you through it.

Step-by-step: how to use Rotate PDF

  1. Open Rotate PDF. Go to ilovepdf.cyou/rotate-pdf.
  2. Upload the PDF. Drag and drop the file. Multi-page PDFs are fully supported.
  3. Choose the rotation angle. Pick 90° clockwise, 180°, or 270° (counter-clockwise) from the dropdown.
  4. Specify which pages. Type all to rotate every page, or a comma-separated list like 1,3,5 for selected pages only.
  5. Process and download. Click Process — the rotated PDF saves as ILovePDF-rotate.pdf.

Why use Rotate PDF?

  • Permanent fix. Unlike viewer-only rotation, the new file is saved with the correct orientation embedded.
  • Per-page control. Rotate only the pages that need it instead of the entire document.
  • Lossless. Pages are rotated in metadata — nothing is re-rendered or compressed.
  • Print-ready. The output prints correctly on any device without anyone needing to manually reorient.

Common use cases

For Office scanning

Multi-page receipts and contracts that came out the office scanner sideways become a clean, properly-oriented document.

For Smartphone scans

PDFs created from phone camera scans (CamScanner, Notes app) often have inconsistent orientation. Fix the whole batch in one pass.

For Compliance archiving

Standardise the orientation of records before adding them to a long-term archive so future readers don't struggle.

Pro tips for the best results

1. Combine with Crop

After rotating, use Crop PDF to trim any black borders left by the scanner.

2. Rotate before merging

If you're going to merge multiple PDFs, fix orientations first so the final file is consistent.

3. Use 270° for upside-down scans

180° flips an upside-down page upright; 270° handles pages scanned in the wrong landscape direction.

4. Re-OCR if rotated

If your PDF was OCR'd before rotation, run OCR PDF again — text positions change with rotation.

Frequently asked questions

Will rotating shrink quality?

No. Pages are reoriented at the metadata level — image pixels and vector data are untouched.

Can I rotate by an arbitrary angle?

Only multiples of 90° are valid in the PDF spec. For arbitrary angles you'd need to convert to image first using PDF to JPG.

Is the rotation reversible?

Yes — just rotate by the inverse angle (e.g. 270° to undo a 90° rotation).

Does it work on encrypted PDFs?

Use Unlock PDF first if you own the file, then rotate the unlocked copy.
